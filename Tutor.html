Cara Generate AAB Flutter & Upload ke Play Store

Panduan lengkap dari build hingga publish ke Google Play

1️⃣ Generate Keystore

Jalankan perintah berikut di terminal:
keytool -genkey -v -keystore upload-keystore.jks -keyalg RSA -keysize 2048 -validity 10000 -alias upload

Simpan file upload-keystore.jks ke:

android/app/
---

2️⃣ Buat File key.properties

Buat file di:

android/key.properties

Isi dengan:

storePassword=PASSWORD_KAMU
keyPassword=PASSWORD_KAMU
keyAlias=upload
storeFile=upload-keystore.jks
---

3️⃣ Konfigurasi build.gradle

Edit file:

android/app/build.gradle

Tambahkan konfigurasi berikut:

signingConfigs {
    release {
        keyAlias keystoreProperties['keyAlias']
        keyPassword keystoreProperties['keyPassword']
        storeFile file(keystoreProperties['storeFile'])
        storePassword keystoreProperties['storePassword']
    }
}

buildTypes {
    release {
        signingConfig signingConfigs.release
        minifyEnabled false
        shrinkResources false
    }
}
---

4️⃣ Build File AAB

flutter clean
flutter pub get
flutter build appbundle --release

File AAB akan tersedia di:

build/app/outputs/bundle/release/app-release.aab
---

5️⃣ Upload ke Google Play Console

Langkah Upload:

  • Login ke Google Play Console
  • Klik Create App
  • Isi Store Listing (deskripsi, icon, screenshot)
  • Masuk ke Production → Create New Release
  • Upload file .aab
  • Isi release notes
  • Kirim untuk review
---

6️⃣ Update Versi Aplikasi

Edit file pubspec.yaml:

version: 1.0.1+2

Lalu build ulang:

flutter build appbundle --release
---

⚠️ Tips Agar Tidak Ditolak

  • Gunakan targetSdk terbaru
  • Jangan pakai debug keystore
  • Pastikan tidak crash saat dibuka
  • Wajib punya Privacy Policy
